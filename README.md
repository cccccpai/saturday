## 快速概览

首先你会注意到文件夹下没有`components/`, `pages/`, `stores/`，`styles/``src/`,文件夹。

你将会看见这几个文件夹，称之为模块

- `core/`
- `users/`

你可以在每个模块中添加以下文件夹，而无需任何手动导入，

- `components/`
- `modules/`
- `pages/`
- `stores/`
